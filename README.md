# brazo_robt_arduino
Robotic arm with Arduino controlled by infrared

- Material:
 * Arduino UNO R3
 * Proto Shield
 * Motor paso a paso
 * 4 Servomotores
 * Jumpers Macho-Macho, Macho-Hembra
 * Bateria 9 V
 * Estructura del brazo impresa 3D
 * Interruptor
 * Control y Receptor infrarrojo
